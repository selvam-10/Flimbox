## 🎬 FilmBox

A simple web application built using Flask and MySQL to display a movie listing, details, and an admin interface for adding plot/cast information.

## 🛠️ Technology Stack

* Backend: Python (Flask)
* Database: MySQL
* Frontend: HTML5
* Styling: CSS3
